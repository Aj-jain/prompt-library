# Debugging Prompts

Prompts for troubleshooting, error analysis, and fixing bugs.

## Available Prompts

- `debugging_error-trace_general.md` — Analyze error traces
- `debugging_bug-investigation_general.md` — Investigate and fix bugs
- `debugging_performance-issue_general.md` — Debug performance problems

## How to Use

1. Select a prompt from this directory
2. Copy the template
3. Add your error details or logs
4. Paste into your AI tool for debugging assistance
