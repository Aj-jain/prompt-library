# Stock Market Analysis Prompts

Prompts specialized for stock market analysis, trading strategies, financial forecasting, and investment decision-making.

## 📊 Purpose

This directory contains prompts designed to help:
- Analyze stock market trends and patterns
- Evaluate investment opportunities
- Develop trading strategies
- Forecast market movements
- Assess financial health of companies
- Generate investment reports

## 📋 Available Prompts

- `stock-market-analysis_technical-analysis_support-resistance.md` — Identify support and resistance levels

## 🚀 How to Use

1. **Browse the prompts** by category above
2. **Copy the template** from the relevant prompt file
3. **Fill in your data** — stock tickers, timeframes, financial figures, etc.
4. **Paste into your AI tool** (ChatGPT, Claude, etc.)
5. **Get analysis and insights** for your investment decisions

## 💡 Naming Convention

All prompts follow this structure:

```
stock-market-analysis_{analysis-type}_{specific-focus}.md
```

**Examples:**
- `stock-market-analysis_technical-analysis_chart-patterns.md`
- `stock-market-analysis_fundamental-analysis_earnings.md`
- `stock-market-analysis_trading-strategy_day-trading.md`

## 📝 Prompt Template

Each prompt includes:

```markdown
# [Prompt Title]

## Purpose
What this analysis does and when to use it.

## Context
- Best use cases
- Required data
- Key considerations

## Template
[The actual prompt with {placeholders}]

## Example
**Input:** {Example with actual stock data}
**Output:** {Expected analysis result}

## Notes
- Important tips
- Common pitfalls to avoid
- Related prompts or strategies

## Tags
`tag1` `tag2` `tag3`
```
