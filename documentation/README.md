# Documentation Prompts

Prompts for writing, structuring, and organizing technical documentation.

## Available Prompts

- `documentation_api_quickstart.md` — Create API quickstart guides
- `documentation_user-guide_general.md` — Write user guides
- `documentation_tutorial_step-by-step.md` — Create step-by-step tutorials

## How to Use

1. Select a prompt from this directory
2. Copy the template
3. Fill in your specific content details
4. Paste into your AI tool for assistance
