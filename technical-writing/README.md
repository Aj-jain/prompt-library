# Technical Writing Prompts

Prompts for creating technical content like blog posts, whitepapers, and case studies.

## Available Prompts

- `technical-writing_blog-post_general.md` — Write technical blog posts
- `technical-writing_whitepaper_research.md` — Create whitepapers
- `technical-writing_case-study_general.md` — Develop case studies

## How to Use

1. Select a prompt from this directory
2. Copy the template
3. Fill in your topic details
4. Paste into your AI tool for writing assistance
