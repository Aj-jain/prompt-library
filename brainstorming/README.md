# Brainstorming Prompts

Prompts for ideation, creative thinking, and problem-solving.

## Available Prompts

- `brainstorming_feature-ideas_general.md` — Generate feature ideas
- `brainstorming_solutions_problem-solving.md` — Find solutions to problems
- `brainstorming_design-concepts_general.md` — Develop design concepts

## How to Use

1. Select a prompt from this directory
2. Copy the template
3. Describe your topic or problem
4. Paste into your AI tool for creative assistance
