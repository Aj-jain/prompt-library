# Code Review Prompts

Prompts designed for analyzing code, reviewing pull requests, and evaluating code quality.

## Available Prompts

- `code-review_pr-analysis_performance.md` — Analyze PR for performance issues
- `code-review_security-review_general.md` — Security-focused code review
- `code-review_best-practices_style.md` — Code style and best practices review

## How to Use

1. Select a prompt from this directory
2. Copy the template
3. Fill in your code or PR details
4. Paste into your AI tool for analysis
